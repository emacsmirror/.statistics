* Imported from the Emacsorphanage

The upstreams of these packages use some version control system
other than Git or Mercurial.  As you can see there are very few
such packages, which is why support was removed from Melpa.
Not having to support five vcs for the benefit of a handful of
packages makes it more appealing to improve the remaining code.

- ~shimbun~ isn't listed below because it is part of the ~w3m~ repository.

#+BEGIN_SRC emacs-lisp
  (epkg-with-org-header ("Package" "Vcs") ; "Last commit" "Commits last year")
    (-keep (pcase-lambda (`(,name ,vcs))
             (let ((default-directory
                     (if (member vcs '("darcs" "fossil"))
                         (format "~/git/emacs/import/%s/%s/%s_git/"
                                 vcs name name)
                       (format "~/git/emacs/import/%s/%s/" vcs name))))
               (and (melpa-get name)
                    (list name vcs
                          ;; (magit-git-string
                          ;;  "show" "-q" "--date=relative" "--format=%cd")
                          ;; (length
                          ;;  (magit-git-lines
                          ;;   "log" "--since=1 year ago" "--oneline"))
                          ))))
            emir--other-vcs))
